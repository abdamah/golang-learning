## Exercises

1. **Hot day** — `temperature := 30`. If greater than 25 print `Hot day`, else `Nice day`.
2. **Password** — `password := ""`. If empty print `Password required`, else `Password OK`.
3. **Port** — `port := 8080`. If between 1 and 65535 print `Valid port`, else `Invalid port`.
